# IHBB Simulator Web

A fast, browser-based IHBB practice app for running tossup drills, buzzing in, tracking mistakes, and reviewing progress.

## Live App (Vercel)
👉 **Vercel URL:** `ihbb-simulator-web.vercel.app`

> Replace the URL above with your real Vercel deployment link.

## What This Project Is
IHBB Simulator Web is designed to feel like real competition practice while staying lightweight and easy to use in a browser. It supports solo prep, targeted review, and repeat practice on weak areas.

## Core Features
- **Game-like practice flow** with read-aloud style prompts, buzz timing, and timed answering.
- **Smart filtering** by Region, Era, and Source so you can train specific categories.
- **Instant grading** with local matching and optional DeepSeek-assisted evaluation.
- **Review system** with session history and a wrong-answer/SRS workflow.
- **Library tools** to import/export and manage question sets.
- **Question pipeline support** for converting raw source text into app-ready `questions.json`.

## Why Use It
- Practice in a realistic IHBB rhythm.
- Turn mistakes into repeatable drills.
- Keep everything in one place: practice, review, and question management.

## 项目定位
这个项目强调“直接开始训练”，尽量还原比赛练习节奏，同时把练习、复盘和题库管理放在同一个界面里。

## 主要功能
- **接近实战的练习流程**：读题 + 抢答 + 限时作答。
- **多维筛选**：按 Region / Era / Source 定向训练。
- **即时判分**：本地判分，并支持可选 DeepSeek 辅助判分。
- **复盘系统**：历史记录与错题 SRS。
- **题库管理**：支持导入/导出与题集管理。
- **数据构建能力**：可将原始文本处理为 `questions.json`。
